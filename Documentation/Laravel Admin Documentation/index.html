A) Overview - top

This project is a E-commerce Laravel Admin panel and API.We have build this admin panel for who want to create laravel E-commerce admin panel and Rest API for application.

Some of the main Technologies used are

  1. PHP
  2. Laravel 8.x
  3. Bootstrap 4.x

B) System Requirements - top

To installing Otrix Commerce Laravel Admin, Your web server must be running PHP 7.2 or higher and Mysql 5.7 or higher. We have tested it with Windows, Mac and Linux but for optimum performance we will recommend to use Linux with Apache webserver. Below are a list of items you should ensure your host can comply with.

  • PHP 7.3+
  • MYSQL 5.7+
  • Composer 2.x+
  • MBString Extension
  • MYSQLi Extension
  • CURL Extension (recommended)

* In most hosting accounts, these extensions are enabled by default. But you should check with your hosting provider.

C) Installation - top




  1. Create Database:Create new database on your web server and import otrixcommerce_db.sql file under the Otricommerce Database folder as shown below




  2. Now upload otrixcommerce laravel zip file on your webserver using FTP/Cpanel. Laravel zip file location: otrixcommercelaravel > Code > laraveladmincode

  3. Now open command promte and navigate to your project folder (we recommend to use Putty)
  4. Then run below command
    5. composer install
  5. And run this command after composer install
    6. php artisan key:generate
  6. After Successfully installtion laravel open your .env file and set your database confgration.


    7. Congratulations! your laravel application Successfully installted and ready to use

D) Run Your Project - top

Open Admin Panel

After successful login you will reach to Admin Dashboard.



Admin Panel consist 21 modules listed below -

  1. Catalog > Categories - For all Categories and sub-categories
  2. Products - Product Management module create/update/delete product
  3. Coupon - Create Coupon for customer with percentage/fixed amount
  4. Option - Create option here like size.color etc
  5. Manufactureres - Add your product mnufactureres like zara,dell,apple,nike etc
  6. Attribute Group - Here is all type attribute group like your product specification General, Display Feature etc
  7. Review - You can view customer submitted review here
  8. Shipping - Add your shipping method and shipping charges here
  9. Length Class - Crud length class here like your product has length
  10. Weight Class - Crud Weight class here like your product has Weight
  11. Order Status - Crud order status like pending,completed etc
  12. Stock Status - Crud stock status like out of stock,in stock etc
  13. Countries - Crud Country
  14. Tax Rate - Add your tax rates
  15. Customer - Manage your customer like delete,add new etc
  16. Sales > Order - Manage your order like add order history and change order status
  17. User - Manage user for your admin panel use like Editor,sales etc
  18. CMS > Manage Cms - Create pages like terms ,privacy,or any informative
  19. CMS > Deals of the day - Add remove product to your DOD Section
  20. Role&Permission - Manage Role and Permission like create Editor role and assign some specific Permission to him
  21. Design - Change Application banner images from here
